# Instalation process

1. Unzip folder

2. Open CMD prompt

3. Cd to folder

(cd C:\Users\Admin\Desktop\converter)

4. Create python virtual environment

python -m venv venv

5. Activate this environment

venv\Scripts\activate.bat

6. Install needed packages

pip install -r requirements.txt

7. Exit CMD

exit

# Configuration

After this steps change the input and output file name in the config.cfg

# Usage

Clicking on run.cmd should convert the file

run.cmd
